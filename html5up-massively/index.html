Fall 2025

Global Housing Market EDA

This final project analyzes worldwide housing price indices, mortgage rates, and affordability trends from 2010–2020. The project includes animated bar charts, time-series visualizations, and a global choropleth map designed to illustrate how national housing markets change over time.

The goal is to translate complex datasets into intuitive visual summaries that highlight inequality, volatility, and long-term housing patterns.

Fall 2025

DC Census Portfolio

The DC Census Portfolio is a course project analyzing demographic, economic, and housing variables across Washington, D.C. neighborhoods using publicly available Census data.

This project focuses on learning the fundamentals of data cleaning, summary statistics, exploratory visualization, and geographic breakdowns. Through plots, tables, and map-based graphics, it translates raw Census numbers into accessible visual patterns that help readers understand how characteristics like age, income, and housing costs vary across the District.

This assignment serves as a foundational exercise in working with real-world public datasets and communicating insights clearly to non-technical audiences.

About This Portfolio

• Turning messy data into structured insight
• Using Python and Jupyter to build transparent analysis pipelines

• Connecting data to meaningful social and spatial questions
